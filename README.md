# Interactive Calendar Component

## 📌 Overview
This project is a responsive, interactive calendar component built using React. It is inspired by a wall calendar layout and focuses on clean UI, usability, and state management.

## 🚀 Features

- 📅 Dynamic calendar grid generation
- 🔵 Date range selection (start, end, and in-between highlighting)
- 📝 Notes system:
  - General notes
  - Notes attached to selected date ranges
- 💾 LocalStorage persistence
- ⬅️➡️ Month navigation
- 🔴 Today highlight
- 📱 Fully responsive (desktop + mobile)
- ✨ Smooth animations and hover effects

## 🧠 Technical Decisions

- Used React state for managing calendar and selection logic
- Implemented custom date range selection logic without external libraries
- Used localStorage for persistence (no backend required)
- Component-based architecture for better maintainability
